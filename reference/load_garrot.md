# Load and process the Garrot dataset

This function loads and processes the "Garrot" dataset from a predefined
external source. It validates the file and columns, applies
transformations, and standardizes the data structure.

## Usage

``` r
load_garrot()
```

## Value

A processed `data.frame` with standardized columns and integrated
species codes.

## Examples

``` r
if (FALSE) { # \dontrun{
processed_garrot <- load_garrot()
} # }
```
