# Load and process the Atlantic Colonies dataset

This function loads and processes the "Atlantic Colonies" dataset from a
predefined spatial data source. It validates the file, applies
transformations, and integrates coordinates with census data.

## Usage

``` r
load_atlantic_colonies()
```

## Value

A processed `data.frame` with standardized columns and integrated
spatial data.

## Examples

``` r
if (FALSE) { # \dontrun{
processed_colonies <- load_atlantic_colonies()
} # }
```
